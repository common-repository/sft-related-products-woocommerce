== Installation ==
1. Upload the sft-related-products-woocommerce plugin folder to your /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Click on the \'Settings\' link or in the WordPress dashboard click on Related Products menu
